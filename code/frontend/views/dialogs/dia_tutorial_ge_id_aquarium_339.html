Hier siehst du dein aktuell noch leeres Aquarium. Jedes 3. Tag schaltet neue Pflanzen oder Fische frei, die dann für dich sichtbar im Aquarium werden.

Neben diesen Elementen kannst du zusätzlich noch 3 Überraschungen freischalten. Unter dem Aquarium findest du eine kleine Anzeige, die dir einen Hinweis darauf gibt, wie viele Tags du noch bis zur nächsten Überraschung brauchst.

Solltest du am Ende eine bestimmte Anzahl an Tags erreicht haben, so schaltest du eine weitere besondere Überraschung frei.

Das Aquarium wird nach der Einführung zurückgesetzt.
