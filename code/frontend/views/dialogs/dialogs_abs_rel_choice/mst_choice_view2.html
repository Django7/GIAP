Wähle deine Tagging Aufgabe!

Während des Tutorials hast du zwei verschiedene Versionen der Tagging Aufgabe kennengelernt. Du hast nun die Möglichkeit auszuwählen, welche Version du während der Studie nutzen möchtest.

Klicke "Erste Version verwenden" wenn du die erste Version der Rangliste verwenden möchtest, oder klicke "Zweite Version verwenden" wenn du die zweite Version der Bestenliste verwenden möchtest.

Sobald du dich entscheidest, startet die Studie.
